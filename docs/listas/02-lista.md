# Lista 02 — Missão: Dando Estilo às Caixas

!!! abstract "Briefing da Missão"
    Aplique a [Aula 02](../aulas/02-aula.md): CSS externo, seletores e **Box Model**. Deixe suas páginas com cara de gente.

## 🎯 Exercícios

??? abstract "Missão 1 — Cartão de visita"
    Estilize um `<article>` como cartão: largura fixa, `padding`, `border`, `border-radius` e `box-shadow`. Use `box-sizing: border-box`.

??? abstract "Missão 2 — Explorando o Box Model"
    Três caixas com o mesmo `width` mas `padding`/`border` diferentes. Abra o **DevTools** (aba *Computed*) e compare o box model de cada uma (inclua um print no README).

??? abstract "Missão 3 — Paleta e tipografia"
    Defina cor de fundo, cor de texto com bom contraste, uma fonte do Google Fonts e `line-height: 1.6`. Justifique as escolhas em comentários no CSS.
