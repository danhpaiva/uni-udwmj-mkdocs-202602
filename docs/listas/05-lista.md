# Lista 05 — Missão: Olhar Crítico de UX

!!! abstract "Briefing da Missão"
    Coloque o chapéu de designer ([Aula 05](../aulas/05-aula.md)): avalie interfaces com as **heurísticas de Nielsen** e pense no usuário.

## 🎯 Exercícios

??? abstract "Missão 1 — Caça às heurísticas"
    Escolha um app/site que você usa. Liste **3 boas práticas** e **3 problemas**, citando o número da heurística de Nielsen em cada caso.

??? abstract "Missão 2 — Persona"
    Crie uma persona (nome, idade, objetivos, frustrações, contexto) e explique como ela influenciaria uma decisão de design.

??? abstract "Missão 3 — Redesenho justificado"
    Encontre um formulário confuso e proponha uma versão melhor (pode ser rascunho/imagem), justificando cada mudança com a heurística correspondente.

## 🚀 Passo a passo da entrega

1. Crie o repositório **público** **`uni-udwmj-lista05`**.
2. Escreva a análise em Markdown (`README.md`) com as imagens em `img/`.
3. Suba o conteúdo:
   ```bash
   git init
   git add .
   git commit -m "docs: resolve lista 05"
   git branch -M main
   git remote add origin https://github.com/SEU_USUARIO/uni-udwmj-lista05.git
   git push -u origin main
   ```
4. **Entregue o link do repositório** no **Google Classroom**.

!!! check "Checklist de qualidade"
    - [ ] Cada problema/boa prática cita o **número** da heurística.
    - [ ] Persona com objetivos e frustrações realistas.
    - [ ] Redesenho justificado por evidências, não por gosto pessoal.
    - [ ] Repositório público `uni-udwmj-lista05` e link no Classroom.
