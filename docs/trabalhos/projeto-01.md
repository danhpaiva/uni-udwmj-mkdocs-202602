# Projeto 01

!!! abstract "Resumo"
    Criar um pequeno site de documentação usando MkDocs Material, aplicando o que foi visto na [Aula 01](../aulas/01-introducao.md).

## Enunciado

Escolha um tema de sua preferência e monte um site com, no mínimo:

- Uma página inicial (`index.md`) apresentando o tema.
- Duas páginas de conteúdo organizadas no menu lateral (`nav`).
- Pelo menos um **admonition**, uma **tabela** e um **bloco de código**.

## Critérios de avaliação

| Critério                     | Peso |
| :--------------------------- | :--- |
| Organização e navegação      | 30%  |
| Uso correto de Markdown      | 30%  |
| Recursos do Material (caixas, abas, código) | 20% |
| Publicação no GitHub Pages   | 20%  |

## Entrega

1. Suba o projeto para um repositório no GitHub.
2. Ative o deploy pelo GitHub Actions (veja o [Guia do Professor](../setup.md)).
3. Envie o **link do site publicado** pela plataforma da disciplina.

!!! danger "Prazo"
    Entregas após a data limite não são aceitas pelo sistema. Não deixe para o último dia.
